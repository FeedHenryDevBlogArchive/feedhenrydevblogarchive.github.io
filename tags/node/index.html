If you have been running into issues with your Node code today, it is probably due to the new version of Express (4.0) which is now in NPM. In a FeedHenry context you may be seeing that all of your cloud calls respond with the same message - “Your Cloud App is Running”. This will happen if your package.json contains a wildcard “*” for the Express version. So when you deploy your app, it automatically picks the latest version from NPM.
