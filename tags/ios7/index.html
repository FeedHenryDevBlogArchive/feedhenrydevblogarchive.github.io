A reminder to all developers about the iOS7 SDK

04 Feb 2014, 10:04
build service / ios7

One of the most popular features in the FeedHenry Platform is our Cloud Binary Build Service. This enables you to build Hybrid iOS, Android, Windows Phone and Blackberry Apps from within our web-based Studio rather than needing all of the tools on your own machine. If you are an iOS developer, you are probably aware that all apps submitted to the App Store after Feburary 1st have to be built with the iOS 7 SDK.
