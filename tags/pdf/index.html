Server side PDF generation with Node.js

21 Feb 2014, 16:45
handlebars / node / node.js / pdf / phantomjs

Overview of what was required A solution in Node.js for dynamically generating a PDF version of submitted form data. The form data is stored in a mongo database, backed by some mongoose models. What this boils down to is we have a collection (JSON array) of forms (JSON objects) with a number of key-value pairs. These key-value pairs have to be written out to a PDF file. The layout of the PDF file could be tweaked later to present the data in the best possible way.
