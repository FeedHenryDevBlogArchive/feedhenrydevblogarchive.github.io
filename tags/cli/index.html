A summary of the FeedHenry 3 changes in our fhc command-line tool

24 Jun 2014, 17:26
cli / feedhenry3 / fhc / node / node.js / npm

We develop everything in FeedHenry API-first ,which means our command-line tooling is just as powerful as our Web UIs, with the added advantage of automation and scriptability. The main FeedHenry CLI is called fhc and it is how most developers interact with the platform. Version 0.30.x has had quite a few changes (to put it mildly!) for FeedHenry 3 and we’ll summarise those changes here. fhc will continue to work with both FeedHenry 3 and previous versions of the platform for the foreseeable future.
