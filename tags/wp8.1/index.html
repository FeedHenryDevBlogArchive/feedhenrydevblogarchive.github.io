Announcing the FeedHenry C#/.NET Native SDK for Windows Phone and Xamarin

18 Jun 2014, 16:25
.net / pcl / sdk / windows phone / wp8 / wp8.1 / Xamarin

If you are a .NET developer, you can now use our .NET SDK to develop native cross-platform cloud-connected apps. It can be used to develop Windows Phone 8+ apps (using Visual Studio) as well as iOS and Android apps (using Xamarin Studio). The SDK is open source and available here on GitHub. It can be downloaded via the NuGet .NET package management system here. We have an introduction to the .NET SDK, and all of our SDKs on our docs site.
